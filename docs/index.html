まもる棚 サポートページ

まもる棚は、家庭内の備蓄品、食品、日用品、防災用品などの期限管理を補助するアプリです。 このページには、ストア提出時に必要なサポート、プライバシーポリシー、利用規約へのリンクをまとめています。

ストア提出用URL

  • プライバシーポリシー: privacy.html
  • 利用規約: terms.html
  • サポートURL: support.html

GitHub Pagesで公開した後、ストアには https://<ユーザー名>.github.io/<リポジトリ名>/privacy.html のようなHTTPSの公開URLを入力してください。
